# Clipboard Manager

Менеджер буфера обмена для macOS. Живёт в строке меню и хранит историю скопированного текста.

## Возможности

- **История** — до 50 последних текстовых копирований с временем.
- **Горячая клавиша** — **⌘⇧V** открывает и закрывает окно истории из любого приложения.
- **Запуск при входе** — опциональная автозагрузка (macOS 13+).
- Окно открывается по клику на иконку в меню или по горячей клавише; закрывается при клике вне окна.

## Использование

Выбери элемент в списке (мышь или стрелки), нажми Enter или двойной клик — текст снова попадёт в буфер и в начало истории. Кнопка очистки удаляет всю историю и содержимое буфера.
